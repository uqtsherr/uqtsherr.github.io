Blind Source Separation of Speech

I developed a Real Time Blind Source Separation System for GN Resound. This system would separate up to 10 Voices speaking over each other within a reverberant environment (4mx4m room). It used frequency domain Independent Component Analysis (ICA), along with a backprojection algorithm to obtain an average +17dB improvement in SNR when compared to the inital recordings. Practially tested (the audio clips are from a real world test), with an end-end delay of ~100ms. developed in Matlab, ported to C

EEG Source Localisation

EEG Source localisation is a powerful tool for diagnosis of a variety of neurological conditions. However, to do it right, an anatomically accurate conduction model of the subject is key, and the gold standard is based off MRI imagery. However, obtaining an accurate model for an infant is challenging, firstly because critical anatomical features, for example the skull are often not resolved in the image due to how thin they are. Secondly infants are prone to wriggling about, introducing movement artifacts and noise.
My work takes contempoarary MRI segmentation algorithms and adds anatomical constraints around feature thickness to ensure models maintain anatomical accuracy even under adverse imaging conditions.

Automated Seizure Detection Via Deep Learning

Seizure burden is a strong marker for adverse neurological outcomes. It is a powerful tool in quantifying injury during acute neurological trauma. However, accurately classifying seizure burden is a very time intesive task. An EEG expert needs to view the entirety of the EEG record to identify all seziures that are present, which takes roughly 1/3 of the original recording time . Recordings in clinical trails can exceed 72hrs, a huge burden on the scientists doing the markup. I have developed an SVM classification system which obtains 98.6% accuracy. This is in line with modern alogrithms. I have also developed a CNN classification system with similar performance. I am investigating the potential for LSTM's to further improve on the performance.

Ship Motion System

This project involved building a Data Acquisiton System using the NI Compact RIO platform, building strain + accelerometer sensor nodes and installing the whole system on a bulk tanker. In addition I have developed a bending moment estimation algorithm, which takes the strain gauge sensor data, fits that data to the FEM analysis from the ship design team, and estimates the first order bending moment experienced by the hull. This provides real world fatigue data for the owners asset management team.
